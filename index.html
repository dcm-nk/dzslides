What do I do?

I work for Mozilla specifically focused on making Firefox more extensible, more hackable, more customizeable. This project is called Jetpack, and although I'm not here to talk about Jetpack specifically, what I'm going to show you was built with Firefox and Jetpack.

Enter Firefox OS

Firefox OS, along with our mobile web app marketplace for Android lets you build web apps using web standards. We're shipping hardware now, we've got some demo devices here tonight.

The mobile web isn't just about putting a crappy, slow browser on a phone. That's old. We're building a mobile OS for the web.

Build for the Open Web. Deploy to:

  • Firefox OS
  • Android ( Firefox Apps )
  • Android / iOS ( PhoneGap ))
  • Desktops running modern browsers

We're not saying Firefox OS will have Billions of users. We're saying that it doesn't matter, the web already has billions of users. If you build a great web app - that's your market. Every computer in a web cafe, every iPad, every Android phone.
